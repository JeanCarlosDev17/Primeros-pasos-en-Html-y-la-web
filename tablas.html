Tablas en Html5

  • table Indica el inicio y el cierre de la tabla
  • tr (Table Row) Representa una nueva fila en la tabla
  • td (Table Data Cell) Representa una celda de datos
  • th Celda de cabecera , resalta el texto
  • caption Añade un titulo sobre la tabla

    Etiquetas de tablas para agrupar

  • thead Agrupa las celdas de cabecera
  • tbody agrupa las filas de cuerpo que contienen los datos
  • tfoot Agrupa las filas del pie de la tabla normalmente son las que contienen los datos de los totales y conclusiones

Combinar celdas de las tablas

Mediante el uso de atributos colspan y rowspan de las etiquetas de tablas se puede indicar cuanto espacio va a ocupar una celda

  • Colspan este atributo indica el numero de celdas que ocupa Horizontalmente
  • Rowspan este atributo idica cuantas celdas de espacio ocupa Verticalmente
